Lettings Folder structure

This is for all documents that are edited on a daily basis. New instructions, valuations, faxes, accounts, invoicing and so on. The entire folder structure will be synchronised overnight. Each branch has it's own folder structure, which should only be edited or added to in that branch (note 1). All branches can view the files from any branch.

Any incoming faxes (from the InterFax email service) should be saved in the appropriate folder on arrival.

Templates:

This folder structure does not include templates. All templates (i.e. non-editable documents) are created and stored at head office, then farmed out to the other offices each night. If you require any changes or additions to the templates folder they should be sent to Mark. You should never save anything in the templates folder.

Unsuitable files:

Because this process is run over a broadband connection, it takes some time. It is important therefore to keep the folders as small as possible. You should only be saving Word Documents, Excel spreadsheets, Adobe Acrobat PDF and similar document files in this folder structure. No large files (photos and floor plans for example) should ever be saved here (these are synchronised but as a separate job).

Naming Convention:

Documents and files, where they relate to a property, should follow a basic naming convention of Street Name, House Number (e.g. Shad Thames, Spice Quay, 30b or Silvester Road, 66) so the folders can be sorted by name and you can find what you want quickly.
